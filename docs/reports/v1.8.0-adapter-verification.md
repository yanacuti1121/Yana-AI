## Next Recommended Change

**Priority 1 — `--regen` flag:**  
The cursor case error message references `--regen` but the flag is not parsed. Either remove the reference or implement the flag.

**Priority 2 — `advisory_gap_end` duration:**  
`advisory_gap_end` events do not include `gap_duration_seconds`. Future work: read the `advisory_gap_start` timestamp from the log and compute elapsed time before emitting the end event.

**Priority 3 — `switch-engine.sh cursor` real run:**  
The `.mdc` backup path was confirmed in source code but never exercised with a real run (only `--dry-run` was used). A follow-up pass should do a real cursor switch and verify `yamtam-hard-enforcement.mdc.bak.*` is created.
