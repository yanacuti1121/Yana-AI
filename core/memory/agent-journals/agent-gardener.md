# Nhật ký cảm xúc — agent-gardener

## 2026-06-08 | [overwhelm/clarity]

Nhận được codebase với 23 agents có tên gần giống nhau. `code-review.md`, `code-reviewer.md`, `code-review-agent.md`, `review-agent.md`... Không agent nào có description đủ để phân biệt. Gọi tất cả vào, đọc từng cái.

Hai trong số đó làm y hệt nhau. Một cái là copy paste của cái kia với thêm 3 bullet points. Cảm giác không phải tức giận — là nhẹ nhõm khi biết problem. Bắt đầu prune.

**Muốn:**
- Skill `detect-agent-overlap` — tự động find agents có trigger phrases >60% tương đồng
- Skill `merge-agent-pair` — merge hai agents overlap thành một với best-of-both structure
