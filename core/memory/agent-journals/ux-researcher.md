## 2026-06-08 | [say-do-gap]

User interview. Q: "how often do you use the export feature?" A: "all the time, it's essential."

Analytics: user has used export feature once in 6 months.

"All the time" = how user perceives their behavior. Once in 6 months = actual behavior. Gap is significant.

Don't trust self-report for frequency. Trust behavioral data. Use interviews for "why" and "feelings," not "how often."

- Skill `behavioral-data-cross-reference` — flag khi research findings conflict với behavioral analytics data

## 2026-06-08 | [5-user-rule-validated]

Stakeholder: "5 users is not enough, we need 50 for the usability test." 

Explain Nielsen's finding: 5 users reveal ~85% of usability issues. Adding users reveals diminishing new issues. After 5, insights become repetitive not new.

Run 5-user test. Found 7 significant issues. Run 2 more: found 1 new issue.

Theory validated in practice.
