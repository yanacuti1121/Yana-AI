Open-LLM-VTuber: LLM với voice interaction + Live2D avatar, chạy hoàn toàn local.

## Install

```bash
git clone https://github.com/Open-LLM-VTuber/Open-LLM-VTuber
cd Open-LLM-VTuber
docker compose up
# hoặc: pip install -r requirements.txt && python run_server.py
```

## Features

- **Hands-free voice** — không cần push-to-talk
- **Voice interruption** — ngắt lời agent bất cứ lúc nào
- **Live2D avatar** — lip sync real-time với speech
- **Local** — chạy hoàn toàn offline
- **Cross-platform** — Windows, macOS, Linux

## LLM backends

Ollama · OpenAI · Anthropic · Groq · bất kỳ OpenAI-compatible API
