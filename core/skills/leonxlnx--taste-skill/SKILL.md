Taste Skill là Agent Skills package chống boilerplate UI. Gives AI agents "good taste" — layout, typography, motion, spacing đúng chuẩn thay vì generic slop.

## Install

```bash
npx skills add Leonxlnx/taste-skill
```

## Covers

- **Layout** — grid, spacing, visual hierarchy
- **Typography** — scale, weight, line-height chuẩn
- **Motion** — animation principles, easing, timing
- **Image generation** — reference boards cho web/mobile/brand kits

## Use cases

- Build landing page không trông như template
- UI components với visual polish thực sự
- Generate design reference boards với ChatGPT Images rồi implement bằng Codex/Cursor/Claude Code
