HugoBlox: describe site bằng text → Hugo Chat AI generate pages → deploy anywhere. Mọi thứ là plain Markdown anh own hoàn toàn.

## Quick start

```bash
# Browse templates
# https://hugoblox.com/templates/

# Generate pages with AI
# https://hugo.chat/

# Local dev
git clone https://github.com/HugoBlox/kit
cd kit && hugo server
```

## Use cases

- Landing page / portfolio từ Markdown
- Blog, docs, publications site
- AI-generated page structure + Tailwind blocks
- Self-hosted, deploy anywhere (Netlify, Vercel, GitHub Pages)

## Key features

- Plain Markdown — không vendor lock-in
- Tailwind CSS blocks (Lego-style)
- VS Code CMS extension (Ownable)
- Hugo static site generator
