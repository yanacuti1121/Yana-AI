TablePro là native database client, free và open source, built for developers.

## Install

Download từ https://tablepro.app hoặc:
```bash
brew install --cask tablepro
```

## Supported databases

- PostgreSQL
- MySQL / MariaDB
- SQLite
- MongoDB
- (thêm dần)

## Features

- Native Swift — fast, không Electron
- SQL editor với syntax highlighting
- Data grid với inline editing
- Connection manager
- Free và open source (AGPL v3)
